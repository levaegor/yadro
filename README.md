# Пошаговая автоматизация с использованием Bash/Python, Docker и Ansible

В данном проекте написан скрипт на Python, выполняющий запросы к сервису "https://tools-httpstatus.pickup-services.com/" с дальнейшей обработкой ответов со статус-кодами. В Dockerfile описана сборка проекта на основе образа ubuntu:22.04. Ansible плейбук устанавливает Docker (в нём реализована логика для Linux-дистрибутивов семейства Debian и RedHat), собирает образ и запускает контейнер с выводом логов. 

Для запуска через Ansible необходимо выполнить команду из директории с плейбуком (предполагается, что у Вас есть файл hosts.ini с необходимыми Вам хостами):

```bash
ansible-playbook -i /path/to/hosts.ini playbook.yml --ask-become-pass
```
