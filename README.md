# Пошаговая автоматизация с использованием Bash/Python, Docker и Ansible

В данном проекте написан скрипт на Python, выполняющий запросы к сервису "https://tools-httpstatus.pickup-services.com/" с дальнейшей обработкой ответов со статус-кодами. В Dockerfile описана сборка проекта на основе образа ubuntu:22.04. Ansible playbook устанавливает Docker (в нём реализована логика для Linux-дистрибутивов семейства Debian и RedHat), собирает образ и запускает контейнер с выводом логов.
